秦可卿 之死

页面底部

秦可卿乃梦中仙。红楼梦是一部假故事，真梦幻的书。所以，秦可卿是“真”当中最重要的一部分。世人都尽其所能地研究她的“假”的现实部分，实在是可笑。

宝玉在第五回梦游太虚幻境的时候，刚入梦就 随了秦可卿 至姑苏，看见警幻在姑苏布散相思，并成功地被布散了相思---早把些 邪魔 招入膏肓，随后由警幻带入太虚幻境。这表示，秦可卿梦中仙的第一个作用就是引宝玉入孽海情天。其实，秦可卿的判曲里面有句话叫，宿孽总因情。“情”字既取孽海情天之意，又谐音秦可卿的“秦”姓。

宝玉随警幻去太虚幻境之前就 丢下了秦氏。这已经暗示了秦可卿的速速退场。同样地，对应于非梦境的荒唐故事，秦可卿也完成了让宝玉在她房里歇午觉（对应于随了秦可卿），并因“某种原因”，迅速消失于人世（对应于丢下了秦氏）。当然，秦可卿死了，宝玉也“从梦中听见 说秦氏死了”，吐了口鲜血。

作为梦中仙，秦可卿对王熙凤做了类似的一件事，那就是，在王熙凤的梦里告知她月满则亏的道理，并把烈火烹油的“元春省亲”的“月满”之景构制完毕，然后彻底消失，留下临别赠的两句话，“三春去后诸芳尽，各自须寻各自门”。同样地，对应于非梦境的荒唐故事，秦可卿也完成了让王熙凤（在贾瑞调戏她之前）去探望她，让这个“治家的人”了解“任凭神仙也罢，治得病治不得命。”（除了说她自己的病之外）王熙凤是典型的非梦境的荒唐人物，她从此再也没有和梦幻有任何关系，而成为贾家衰落过程的一面镜子。后来做的“娘娘打发他来要一百匹锦”梦也是江郎才尽，“治不得命”的典型体现。“三春去后”在第七十回探春放风筝，第七十四回惜春矢孤介杜绝宁国府 和 第七十九回迎春误嫁中山狼 均得以实现。所以，继第六十六回 尤三姐演绎完“真梦幻”的部分， 贾迎春最终完成了演绎“假故事”的部分。全书已经完成。所以，第七十九回的故事全部被作者付之一炬，只恐狗尾续貂而已。

宝玉和凤姐的关系是不怎么明显的，除了“姐弟俩”一同在二十五回遭难。一个姓贾一个姓王，真谈不上什么姐弟不姐弟的。如果不能理解和秦可卿的关系，是很难理解得很深入的。凤姐去探病的时候，宝玉也一同前往，再回顾一下那里的宝玉“不觉想起 在这里 睡晌觉梦到“太虚幻境”的事来”吧。

好了，最不重要的部分到了，秦可卿在假故事里面是以什么样的“某种原因”而告别人世退场？

曹雪芹在编这部分的故事应该经历了两个阶段。第一个阶段，强调“梦幻”般的“情”，耻笑“荒唐”的“淫”，所以，他选择描述可卿“淫丧天香楼”，这也是现在最流行的“被公公贾珍逼奸而死”的版本。或许为了满足世人扑风捉影的快乐，或是因为没有现代编辑软件没有来得及删改，曹雪芹留下了“美人自缢”的判画和焦大最有名的“扒灰的扒灰”，对了，还有不定哪个玩意第十三回写的批语“命芹溪删去“遗簪”、“更衣”诸文”。

第二阶段，曹雪芹一改被“逼死”的人物故事结构，回到“（春恨秋悲）皆自惹”的阶段，并精心修改了故事之间的前后承接关系。这在后几十回特别明显，举个简单的例子： 老太妃死了 -> 王夫人不在家 -> 彩霞偷了茯苓霜 -> 五儿错被抓赃 -> 宝玉自承下来 -> 贾环不理彩霞 -> 旺儿媳妇求彩霞做媳妇 -> 赵姨娘为彩霞去贾政那求情 -> 宝玉误以为贾政要考书只能夜读 -> 晴雯帮宝玉说被吓着了 -> 贾母吩咐查赌 -> 懦小姐不问累金凤。为了描述这种土崩瓦解、大厦将倾的故事节奏，这种事件触发形式的方法很好地满足了“结果皆自惹”的目的。换句话说，每次不过都是“万事俱备，只欠东风”，只要把需要的“东风”按点刮来就可以了。凤姐的下红之症越来越厉害，主要描述她的江郎才尽，也是贾府不行的内因所在。其实，各种外因一直没有形成对王熙凤的真正压力（特别让俗人失望的是，她压根就没被休过，呵呵），第七十一回即使有邢夫人的挑拨，也通过鸳鸯之口得到了支持者贾母的了解，可算是“怀才有遇”的幸运儿了。所以，在改编秦可卿之死的阶段，曹雪芹采用了新的方法：秦可卿请王熙凤 -> 宝玉秦钟初次相会 -> 打算一起去家塾 -> 家塾大乱 -> 秦钟去告知秦可卿 -> 秦可卿聪明忒过，致思虑太过 -> 经期每每延长 -> 经期有两个多月没来 以致夭亡。一句话，脂砚斋本回蒙批曾言：虽贫女得居富室，诸凡遂心，终有不能不夭亡之道。

一句话，跟秦钟草木之人，不幸遇到纨绔之友宝玉、温柔之乡家塾而夭亡一样的道理，从养生堂抱养而来的秦可卿就好像神石一样来自荒山野埂，脱胎幸运落到道人所谓的富贵场中、温柔乡里，然而总归是‘美中不足，好事多魔’，瞬间乐极生悲。这也是为什么我们说秦钟、秦可卿的故事暗示了神石第一次投胎入世的不顺和性灵的不断升级。

可笑的是，现在的人挖祖坟的研究方法，楞是从第二阶段现有的成文挖到曾经的手稿，断定了第一阶段的写法。

当然，更可笑的事情，作为假故事的一部分，秦可卿这个仙人是怎么死的，就好像黛玉这个仙人怎么死的一样，本来就不重要，也被作者故意的隐去了，所谓，花开易见落难寻。这么伤心的部分，大家还伸着脖子要满足自己的快感，我想在“真梦幻”中是很难的，死了这条心罢。
